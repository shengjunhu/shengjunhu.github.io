## Python
```
Python是一种解释型语言, 无需编译(PHP、Ruby也是); 
Python是动态类型的, 无需声明变量类型;
Python非常适合面向对象的编程, 支持定义类、继承, 没有访问域(public、private)
Python代码编辑简洁快速，执行效率比编译语言慢, 但允许调用C程序(C++需 extern "C")
```
